class odesolvers.RungeKutta.RungeKutta(f, **kwargs)[source]¶

Bases: odesolvers.ODE.Adaptive

Superclass for explicit 1-level- Runge-Kutta methods: RungeKutta4, Rungekutta2,etc. or 2-levels adaptive Runge-Kutta methods: Dormand & Prince, Cash-Karp or Fehlberg, etc.

Available subclasses are: RungeKutta2, RungeKutta3, RugeKutta1 (Forward Euler), RungeKutta4. DormandPrince, Fehlberg, CashKarp, BogackiShampine, MyRungeKutta (user-supplied RungeKutta methods),

NOTE: This class should be superclass for level-1 methods. A subclass AdaptiveRungeKutta can act as superclass for the level-2 methods. get_order can be in RungeKutta.

advance()[source]¶

Advance the solution, either using one step or a series of adaptive steps, from t[n] tp t[n+1].

calculate_order_1_level(coefficients)[source]¶

Calculate order of 1-level RungeKutta method with help of the known solution u = -e**t.

Coefficients is a square 2d-array (Butcher tableau).

get_order()[source]¶

Return the order of current method, both for non-adaptive and adaptive methods.
